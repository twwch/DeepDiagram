# 告别碎片化绘图：我做了一个 AI 全能绘图助手 DeepDiagram，让 Excel 秒变统计图

作为一名研发负责人，我的日常几乎被各种“图”占满了：
* **早上开会**：需要随手拉个**思维导图**理清混乱的需求；
* **下午写方案**：得画专业的**系统架构图**、**流程图**或**时序图**；
* **复盘汇报**：必须把 Excel 里的枯燥数据变成直观的**可视化统计表**。

过去，我得在两三个工具之间反复横跳（XMind、Draw.io、ECharts...），文件格式互不兼容，调整样式还极其浪费时间。于是我心一横，干脆自己动手，做了一个开源的 AI 绘图平台 —— **DeepDiagram**。

---

### 1. 它是做什么的？

DeepDiagram 不是一个简单的绘图工具，它是一个**基于 Agentic AI 的多模态可视化平台**。

你不再需要去研究怎么连线、怎么调色，甚至不需要学习复杂的语法。你只需要“说”出你的需求，或者把你的文档、数据丢给它，AI 就会自动路由到最合适的 Agent 帮你完成渲染。

### 2. 核心杀手锏：Excel 直接变图表

这是我个人最常用的功能。很多时候我们手里只有一份原始的 Excel 报表，想做个漂亮的可视化分析。

在 DeepDiagram 里，你只需要**上传 Excel 文件**，说一句“帮我分析这些数据并画个统计图”，它就能自动识别数据维度，利用内置的 **ECharts Agent** 生成交互式的精美图表。

### 3. 全场景覆盖，一个工具就够了

除了处理数据，它还集成了开发者和产品人最常用的几大绘图能力：

* **🧠 思维导图**：基于文本输入快速生成分级清晰的 Mind Map，支持实时编辑。
* **🏗 架构图 (Draw.io)**：生成专业级的云架构、网络拓扑图，支持高级画布操作。
* **🧜 Mermaid 引擎**：像写代码一样生成时序图、甘特图、状态图。
* **🎨 精美信息图 (Infographic)**：基于 AntV 引擎，把复杂的商业逻辑变成高颜值的海报。

### 4. 为什么这个项目“不一样”？

* **设计思路可见**：在渲染之前，你会看到 AI 的“思考逻辑（Design Concept）”，它会先告诉你它是如何理解你的架构或数据的。
* **多模型自由切换**：支持 OpenAI、DeepSeek、Claude 等模型，甚至可以连接你本地部署的大模型。
* **完全开源，自主掌控**：项目基于 React 19 + FastAPI + LangGraph 构建，采用 AGPL-3.0 协议。你可以根据自己的需求定制专属 Agent。
